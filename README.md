## üíª Running

To run the game locally run the following commands:

1. `git clone git@github.com:mealeyst/dungeon_delvers.git`
2. `cd dungeon_delvers`
3. 'npm install`
4. 'npm start`

This should load up a local development server that you can hit at
http://localhost:1234/
